# 石榴湾家具超市三合店 — 好评生成器

## 项目简介

为石榴湾家具超市三合店生成真实、多样化、可复制的好评内容，用于抖音和高德地图等平台的线上口碑建设。

## 店铺信息

- **店名**：石榴湾家具超市三合店（原甲林村三合家具家电广场）
- **地址**：镇平县晁陂镇甲林村甲林冷库对面，老312国道路南，石佛寺路口往西三公里
- **卖场面积**：12,000㎡ 大型家具家电卖场
- **核心卖点**：自建卖场、无租金、价格更实惠
- **主营品类**：软床/实木床、布艺/真皮沙发、餐桌茶台、家用电器、衣柜/博古架、办公家具等

## 技术栈

- 纯前端：HTML + CSS + JavaScript
- 二维码：qrcode.js（CDN引入）
- 无后端依赖，可直接部署到 GitHub Pages

## 目录结构

```
新好评/
  index.html          # 主页面
  data.js             # 好评模板数据（1732条维度评价 + 100条安装评价 + 100条开头语 + 100条结尾语）
  css/style.css       # 样式文件（暖色系红棕色主色调）
  js/app.js           # 核心逻辑（生成、复制、二维码、计数）
```

## 运行方法

本地测试：直接浏览器打开 `index.html` 即可，或启动本地 HTTP 服务器。

## 功能清单

1. **多平台好评生成**：支持抖音和高德地图
2. **随机维度**：每次随机选取1-2个评价维度，无需手动选择
3. **随机生成**：每次生成1-2个维度的组合好评，自动加标点符号
4. **一键复制**：复制好评内容到剪贴板
5. **二维码生成**：扫码查看当前好评内容
6. **生成计数**：显示已生成好评条数
7. **响应式布局**：适配移动端（最大宽度520px）

## GitHub Pages 部署步骤

1. 在 GitHub 上创建新仓库（如 `review-generator`）
2. 把项目文件推到仓库：
   ```bash
   git init
   git add .
   git commit -m "初始提交：好评生成器"
   git remote add origin https://github.com/你的用户名/review-generator.git
   git push -u origin main
   ```
3. 进入仓库 Settings > Pages
4. Source 选择 `Deploy from a branch`，Branch 选 `main`，Folder 选 `/ (root)`
5. 保存后等待几分钟，访问 `https://你的用户名.github.io/review-generator/`
